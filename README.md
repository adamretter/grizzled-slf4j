Grizzled-SLF4J: A simple, Scala-friendly front-end to [SLF4J][].
================================================================

The [Grizzled SLF4J][grizzled-slf4j] package provides a very thin
Scala-friendly layer on top of the [SLF4J][] (Simple Logging Fa√ßade for
